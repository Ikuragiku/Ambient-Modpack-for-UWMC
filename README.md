# Ambient-Modpack-for-UWMC

Das Modpack soll das aus meiner Sicht fehlende Ambiente-Gefühl in Minecraft ergänzen und gleichzeitig die Performance verbessern. Es integriert vor allem Ambient-Sounds, besseres Lichtverhalten, dynamische Lichtquellen sowie einige grafische Verbesserungen und hilfreiche Anzeigen ohne unfaire Vorteile.
